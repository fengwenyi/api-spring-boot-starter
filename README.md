# Spring Boot API Starter

## 快速开始

### 添加依赖

**Maven**

```xml
<dependency>
    <groupId>com.fengwenyi</groupId>
    <artifactId>api-spring-boot-starter</artifactId>
    <version>1.1.0</version>
</dependency>
```

**Gradle**

```groovy
implementation("com.fengwenyi:api-spring-boot-starter:1.1.0")
```

### 开启注解

> @EnableApiStarter
